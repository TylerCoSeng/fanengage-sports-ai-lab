# FanEngage – Adaptive AI for Sports Fan Engagement

This repo contains my prototype for **FanEngage**, an AI agent that generates personalized, spoiler-aware messages for basketball fans using IBM watsonx.ai and the Granite 3-8B Instruct model.

# Contents

- `fanengage_notebook.ipynb`  
  - Notebook-based agent implementation and test scenarios.
- `FanEngage_Technology_Statement.pdf`  
  - Written statement on technology (architecture, prompt design, scenarios, limitations).
- `FanEngage_Problem_Solution.pdf`  
  - Problem and solution statement with link to the working prototype.

# How to Run the Notebook

1. Open `fanengage_notebook.ipynb` in Jupyter or IBM watsonx.ai.
2. Set `API_KEY` in the first cell to a valid IBM Cloud API key.
3. Run all cells from top to bottom.
4. Modify the `event` JSON objects in the last cell to generate messages for different scenarios.
